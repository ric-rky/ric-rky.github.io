Churn Prediction for Education Payments

GitHub

Problem: Predict likelihood of payment after collection actions.
Approach: Action-level restructuring, temporal features, Logistic Regression → XGBoost, temporal validation, SHAP explainability.
Result: AUC 0.81; recall 0.72 in top decile; expected 12% reduction in collection costs.

Rental Price Prediction — São Paulo

GitHub

Problem: Estimate rental prices in São Paulo.
Approach: Log-transform (log_rent), VIF, model assumptions validation, regression and tree models.
Result: R² = 0.85; RMSE = R$120 (~3% of true value).

Wine Quality Classification

GitHub

Problem: Classify wine quality based on physicochemical features.
Approach: Pipelines with cross-validation, hyperparameter tuning, class-level metrics, SHAP explainability, MLflow tracking.
Result: F1-macro = 0.71; full reproducibility ensured.
